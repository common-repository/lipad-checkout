Lipad Checkout integrates seamlessly with WooCommerce, offering a frictionless payment experience via Lipad's intuitive checkout interface.

== Description ==

Lipad Checkout integrates the Lipad payment gateway with WooCommerce, enabling quick payments via an express self-checkout interface.

== Installation ==

1. Search and install Lipad Checkout from the marketplace.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the Lipad Checkout settings in the WooCommerce Payment Gateway settings.
4. Start accepting payments with the Lipad express self-checkout.

== Configuration ==

1. Go to WooCommerce > Settings > Payments.
2. Choose "Lipad Checkout" from the list of available payment gateways.
3. Configure the required settings, such as API keys and service codes.
4. Save the changes.
